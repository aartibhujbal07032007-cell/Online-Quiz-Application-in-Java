# Online Quiz Application

The Online Quiz Application is a Java-based desktop application developed using Java Swing and MySQL. It enables students to log in with their name and roll number, participate in a timed multiple-choice quiz, and receive instant results upon completion.

The application features a user-friendly graphical interface, automatic answer evaluation, score calculation, and secure result storage in a MySQL database using JDBC connectivity. Each question is displayed with a countdown timer, ensuring a structured and interactive quiz experience.

This project demonstrates the implementation of Java GUI development, event handling, database connectivity, and object-oriented programming concepts. It is designed to simplify the quiz and assessment process while providing accurate and efficient result management.

## Features

* Student Login System
* Multiple Choice Questions (MCQs)
* Timer-Based Quiz
* Automatic Answer Evaluation
* Instant Score Calculation
* Result Storage in MySQL Database
* User-Friendly Interface
* JDBC Database Connectivity

## Technologies Used

* Java
* Java Swing
* JDBC
* MySQL
* VS Code
